# Laravel e-commerce 

## Start 

* Create .env file 
* Copy / Paste content from .env.example to .env 
* Run the following commands :

```
    composer install
```

```
    php artisan make:storage
```

```
    php artisan refresh --seed
```

```
    php artisan serve
```
